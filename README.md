## 研究方向

### 基于深度学习的压缩图像去伪影算法研究

目前由于图像压缩算法广泛使用，互联网上存在大量压缩图像，若压缩率过大，会导致压缩图像出现一定程度的失真，产生各种压缩伪影，严重影响视觉效果。我们的主要工作就是设计网络来去除压缩伪影，同时一定程度上恢复压缩图像细节信息。
